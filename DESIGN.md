# Design

## Visual Direction

Routsify uses a warm editorial travel language: premium, calm, expert, and commercial. The site should feel like a studio that designs journeys with care, not like a SaaS dashboard or an AI-generated landing page.

## Color System

- Background: warm paper / stone base
- Surface: soft translucent panels with subtle blur
- Primary accent: forest green
- Secondary accents: muted olive and travel-amber highlights
- Text: deep ink green, never flat gray on gray

## Typography

- Display: Fraunces for headlines and key pull quotes
- Body: Manrope for readability and a modern, human tone
- Headings should be large, balanced, and line-broken intentionally
- Avoid overly narrow containers for display copy

## Layout

- Use spacious sections with clear rhythm
- Favor split layouts, editorial panels, and route/timeline structures over repetitive card grids
- Keep the hero clear in the first viewport on mobile and desktop
- Preserve strong CTA hierarchy: WhatsApp first, Fillout second

## Components

- Floating header bar with nav chips
- Premium panels for trust and service explanations
- Route/timeline blocks for process and planning
- Comparison blocks for decision support
- Legal pages with the same visual system, but simpler

## Motion

- Motion should feel restrained and useful
- Reveal sections with subtle opacity and vertical movement
- Add gentle stagger to groups of cards or steps
- Respect `prefers-reduced-motion`

## Content Rules

- Never invent prices, testimonials, destinations, or claims
- Keep Barcelona as the geographic reference
- Keep WhatsApp and the Fillout embed as the main contact paths
- Use verified content from the current site and WordPress XML
